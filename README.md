# uno-mind

An OpenCV based virtual opponent for Uno. The system uses computer vision to track physical cards on a table and pure logic to calculate the bot's next move.

## Current Scope (MVP)

- Tracks standard number cards (0 to 9) placed on the table.
- Uses a logic engine to determine the correct playable card based on color and number matches.
- Operates without machine learning for game decisions, relying entirely on structured game rules.
